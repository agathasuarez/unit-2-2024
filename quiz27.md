## Code
```.py
def count_letters(my_dictionary, msg):
    for letter in msg:
        if letter in my_dictionary:
            my_dictionary [letter]+=1
    return my_dictionary
#test
case1 = count_letters({"w":0,"l":0,"c":0}, "hello world")
```
